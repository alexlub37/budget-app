YoNo Budget V6 intelligente

Cette version garde tout ce qui a été mis en place et devient partageable proprement.

Principe :
- l'app démarre vide pour une nouvelle personne
- tes données restent sur TON téléphone si tu remplaces simplement les fichiers
- les fixes / opérations / catégories sont conservés grâce au stockage local du navigateur

Fichiers à envoyer sur GitHub :
- index.html
- manifest.json
- icon-1024.png
- icon-512.png
- icon-192.png
- icon-180.png
- logo-yono-budget.png

Pour mise à jour :
1. Extraire le zip
2. Upload files sur GitHub
3. Remplacer les anciens fichiers
4. Commit changes
5. Sur iPhone : supprimer l'app puis réinstaller via Safari si l'icône ne se met pas à jour
