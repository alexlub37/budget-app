YoNo Budget V6.2 OFFLINE

Contenu :
- index.html : app complète
- manifest.json : installation PWA
- service-worker.js : mode hors ligne
- icon-1024.png / 512 / 192 / 180 : icônes
- logo-yono-budget.png : logo source

Ce que cette version ajoute :
- l'app peut se relancer hors ligne après avoir été ouverte au moins une fois avec internet
- les données restent stockées sur le téléphone
- les mises à jour de code peuvent être déployées sans retaper les fixes, si on garde la même base locale

Pour mettre à jour GitHub :
1. Extraire le zip
2. Upload files sur GitHub
3. Remplacer les anciens fichiers
4. Commit changes

Important :
- Après installation/mise à jour, ouvre l'app une fois avec internet pour que le mode hors ligne se mette en cache.
- Si l'ancienne version reste affichée, supprime l'app de l'écran d'accueil puis réinstalle-la depuis Safari.
